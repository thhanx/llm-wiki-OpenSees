# OpenSees — DesignSafe User Guide

- 원본: [raw/designsafe-guide-opensees.md](../../raw/designsafe-guide-opensees.md)
- 출처: https://designsafe-ci.org/user-guide/tools/simulation/opensees/opensees/ (curl + 자체 HTML 추출로 확보, WebFetch는 403)

## 핵심 요지

- DesignSafe(NSF NHERI)에서 OpenSees(순차판)를 실행하는 일반 가이드. HPC 작업 제출 방식, 버전 선택, 입출력 파일 처리 등 실무 내용을 폭넓게 다룸(806줄 분량의 상세 가이드).
- OpenSeesSP/MP 전용 가이드와 짝을 이루는 문서로, 순차 실행 기본기를 먼저 다진 뒤 병렬판으로 넘어가는 흐름에 적합.
