# OpenSees — Wikipedia

**OpenSees** (Open System for Earthquake Engineering Simulation)는 National Science Foundation이 후원한 시대(1997-2007)에 Pacific Earthquake Engineering Research (PEER) Center에서 개발한 객체지향 소프트웨어 프레임워크입니다.

## 주요 정보

| 항목 | 내용 |
|------|------|
| 작성 언어 | C++ |
| 운영 체제 | Linux, Windows, Unix-like |
| 웹사이트 | opensees.berkeley.edu |
| 저장소 | github.com/OpenSees/OpenSees/ |
| 라이선스 | http://opensees.berkeley.edu/OpenSees/copyright.php |

## 개요

사용자는 지진에 노출된 구조 및 지반 공학 시스템의 응답을 시뮬레이션하기 위해 유한 요소 애플리케이션을 생성할 수 있습니다. 주요 개발진은 Frank McKenna와 Gregory L. Fenves이며, Michael H. Scott, Terje Haukaas, Armen Der Kiureghian 등이 중요한 기여를 했습니다.

## 라이선싱

교육, 연구, 비영리 단체의 비상업적 목적 사용, 복제, 수정 및 배포가 허용됩니다. 다른 단체는 내부 목적으로만 사용, 복제, 수정이 가능하며 재배포는 불가능합니다.

## 사용

사용자는 Tcl 또는 Python 프로그래밍 언어로 스크립트를 작성하여 애플리케이션을 생성합니다. 소스 코드는 GitHub에서 관리됩니다.

## 약어

"Open System for Earthquake Engineering Simulation"의 올바른 약자 표기법은 OpenSees이며, OpenSEES가 아닙니다. 이는 Tcl의 관례를 따릅니다.

## 역사

프레임워크는 "OpenSees"라는 이름 이전에 PEER 연구 그룹의 시뮬레이션 개발을 참고하여 "G3"라고 불렸습니다. Frank McKenna의 박사학위 논문인 병렬 객체지향 구조 분석이 "G3"의 기초를 형성했습니다. 2024년 3월 현재 버전은 3.6.0입니다.
